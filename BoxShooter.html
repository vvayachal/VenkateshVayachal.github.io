Box Shooter

First Person Shooter

3D First Person Shooter made using Unity for Windows and MacOS.

Features

  1. Collision detection and resolution to detect bullets hitting the objects
  2. Multiple Levels
  3. Shooting the Green boxes before the time runs out takes player to the next level
  4. Shooting the red boxes counts as negative and fails the level
  5. Used particle systems and concepts of animation
