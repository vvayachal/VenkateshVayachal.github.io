Box Shooter

A fun, 3D, first person shooter developed using Unity for Windows and MacOS

Features

  1. Shooting Mechanic
    • Used Unity's built-in Physics library to simulate shooting to shoot bullets(game objects) at targets(Green Boxes)
  2. Scoring
    • Tracked the number of targets shot using collision detection and maintained the player score
    • Added red boxes which negatively impact the score on shooting
  3. Visuals
    • Enhanced player experience by simulating explosion effects using the Particle System in Unity
    • Created immersive player experience by using the animation system in Unity by applying effects to the blocks
