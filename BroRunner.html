Bro Runner

2D platformer designed in Unity using open source assets

A fun 2D platformer game with multiple levels

Features

  1. Parallax scrolling to give the effect of depth to the background
  2. Collision detection and resolution
  3. Animation blending and looping
  4. Menu driven UI for this game to launch multiple levels for it
  5. ‘Player Preferences’ to keep track of the previous high score, player progress and user preferences
    • Only completed levels can be accessed directly from the main menu
