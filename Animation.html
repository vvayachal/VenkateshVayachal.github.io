My work as an Animator for USC Institute for Creative Technologies

Model Created, Rigged and Animated using Blender for a mobile app

  1. Modeling
    • Using Blender sculpting tools, designed a 3D model of a sea lion for use in a mobile app for interface
  2. Rigging
    • Modified the pre-existing shark rig in Blender to fit the needs of the Sea Lion model by modifying the joints, placing the bones on appropriate locations, removing the excess bones
    • Polished the rig by using the weight painting tools in Blender to refine the movements of the sea lion model
  3. Textures
    • Using Blender texture maps, I gave the sea lion model a cartoon look using the toon shader
  4. Animation
    • Using the rigged model and the Blender animation tools, designed animations for various actions like Wave, Flip, and Jump
    • Exported the animations as the 2D Sprite sheets for integrating in the mobile application
