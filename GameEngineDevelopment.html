Game Engine Development

Assignments done for the Game Engine Development class for the Fall 2021 semester

All assignments are done in Prime Engine which is a custom Game Engine written in C++.

Assignments

  1. Changing character behavior depending on other characters in the scene
    • Modified the character (Soldier) behavior to rotate and face the target as it moves continuosly

    Demo

  2. Improve performance using Frustum Culling
    • Created Axis Aligned Bounding Box for the meshes of objects
    • Generated Frustum plane equations for the camera
    • Using the AABB and Camera Frustum, performed frustum culling while gathering draw calls

    Demo

  3. Implemented Physics System
    • Using the AABB generated earlier, performed collision detection and resolution

    Demo

  4. Animation
    • Using provided animations for a character asset, blended between different animation using different techniques.
    • Implemented Full Body Animations, Partial Body Animations and Additive Animations

    Demo

  5. Object Deformation
    • By adding an extra set of UV to the Vertex Buffer, simulated wind effect to achieve object deformation from wind
    • The added UV values stored the height of the current pixel on the object's mesh from the ground. As shown in the demo, the deformation applied depends on the height of the pixel from the ground. Higher the point on mesh, greater the deformation will be.

    Demo
